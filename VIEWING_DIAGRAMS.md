# How to View Mermaid Diagrams

The README.md and DIAGRAMS.md files contain comprehensive mermaid diagrams. Here's how to view them:

## Option 1: GitHub (Recommended)
GitHub natively renders mermaid diagrams. Simply view the files on GitHub:
- [README.md](https://github.com/notdotmarket/notmarket-solana/blob/main/README.md)
- [DIAGRAMS.md](https://github.com/notdotmarket/notmarket-solana/blob/main/DIAGRAMS.md)

## Option 2: VS Code Extensions
Install one of these VS Code extensions:
- **Markdown Preview Mermaid Support** by Matt Bierner
- **Mermaid Markdown Syntax Highlighting** by Bpruitt-goddard

Then open README.md or DIAGRAMS.md and use the preview pane (Cmd/Ctrl + Shift + V).

## Option 3: Mermaid Live Editor
Copy any mermaid code block and paste it into:
https://mermaid.live/

## Option 4: Online Markdown Viewers
Use these online viewers that support mermaid:
- https://stackedit.io/
- https://dillinger.io/
